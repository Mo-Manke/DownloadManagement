# DownloadManagement 功能说明

## 项目概述
该项目是一个基于 Spring Boot 的应用分发与下载管理系统，提供 APK 包管理、下载分发、访问统计、IP 访问控制和管理员后台等功能，适合用于内部或外部分发场景的下载管控。

## 核心功能
### 1. 应用包管理
- 上传 APK 文件，自动保存到本地存储目录并生成唯一文件名。
- 创建应用包记录（名称、版本、标识 flag、文件路径、所属管理员）。
- 支持启用/停用包；同一 flag 只允许一个启用版本。
- 支持更新包信息与文件路径。
- 支持删除包及其对应文件。
- 提供后台页面和 API 两种管理方式。

### 2. 下载分发
- 通过 `/d/{username}/{flag}` 下载指定管理员下的应用包。
- 按 flag 生成下载文件名（如 `flag.apk`），支持中文文件名编码。
- 支持断点续传（HTTP Range 206）。
- 下载事件记录（安装/下载统计）。
- 支持下载速率限制（按 IP 控制）。

### 3. 访问控制与防护
- IP 白名单/黑名单/封禁管理（支持手动封禁、自动封禁）。
- 支持每分钟访问频率限制、并发连接数限制。
- 触发限制可自动封禁，并支持封禁时长设置。
- 访问路径控制：可配置哪些路径启用 IP 控制（默认 `/d/`）。
- 防盗链（Referer 白名单）与简单的每秒访问限制。

### 4. IP 统计与监控
- 记录 IP 访问日志及下载字节数。
- 提供单 IP 与全局访问统计（1h/24h/7d/30d）。
- 支持按小时/按天分布统计与 Top IP 排行。
- 统计写入采用异步批量处理，定时落库。

### 5. 管理员与权限
- 登录鉴权（表单登录 + 记住我）。
- 登录验证码校验。
- 角色区分：超级管理员 / 普通管理员。
- 超管可管理所有用户、审批请求与查看全量数据。
- 普通管理员只能管理自己的包与数据。
- 初始化默认超管账号（`admin / admin123`）。

### 6. 账号申请与审批
- 普通管理员修改用户名需要走审批流程。
- 忘记密码可提交重置申请，由超管审批或直接设置新密码。
- 审批状态支持通过/拒绝。

### 7. 数据看板
- 提供下载量趋势图（按周/按月）。
- 支持超管按用户筛选统计。

## 页面与接口概览（部分）
- 登录页：`/login`
- 管理后台：`/admin`
- 数据看板：`/dashboard`
- IP 控制台：`/admin/ip`
- 用户管理：`/admin/users`
- 账号审批：`/admin/requests`
- 下载地址：`/d/{username}/{flag}`
- 包管理 API：`/api/packages`

## 可配置项（application.yml）
- 数据库连接、上传大小限制
- 存储目录 `app.storage.location`
- 下载速率限制与防盗链
- IP 限制参数（频率、并发、限速、自动封禁）
